About This Workspace

Rev 5 — A browser-based file management environment

What is this?

This is a self-contained workspace that runs entirely in your web browser. It provides a desktop-like environment for creating and managing documents, spreadsheets, and annotated PDFs — all without any server or cloud backend.

Where are my files stored?

All files live in your browser's localStorage — a small storage area on your device tied to this specific browser and web address. Nothing is sent to any server.

⚠️ Important: If you clear your browser data, switch browsers, or use a different device, your files will not be there. There is no sync.

Can I share files between workspaces?

No. Each workspace instance is completely isolated. Files are bound to the specific browser session and domain where they were created. There is no built-in mechanism to transfer, sync, or share files between different workspace instances.

Think of each workspace like a standalone computer. Opening this file in Chrome and Firefox creates two completely separate environments with no connection to each other.

Built-in Tools

Document Editor — Rich text editor with formatting, headings, lists, and quotes.

Spreadsheet — Grid-based editor for tabular data.

PDF Markup — Load any PDF and annotate with highlights, drawings, arrows, stamps (Meets Spec / Clarification / Option), and pinned comments. Export annotated PDFs.

Calculator — Standard calculator with history.

Privacy

This workspace makes zero network requests for your data. All processing happens locally in your browser. No analytics, no tracking, no cloud.
